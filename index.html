This is a Motorcycle Collection Mobile Application. Link found below.

https://github.com/kjesso5/motorcycle_collection_page/blob/main/app-debug.apk

This app is designed to manage the inventory of motorcycles. You can add, delete, and edit motorcycles from a realtime Firebase database. There is also a search feature available from the main activity that lets you run a report on specific types of motorcycles.
